    В таблице находится информация о направлении подготовки.
    Чтобы узнать подробней о направлении подготовки необходимо выделить ее (нажать клавишей мышки), затем на верхней панели нажать на кнопку "Подробно о направлении" (откроется подробная информация).
    Также внизу окна отображается строка информации о данном направлении подготовки.
    Чтобы занести данные нового абитуриента необходимо выделить направление подготовки и нажать на верхней панели кнопку "Добавить абитуриента" (откроется панель добавления). 
    Для работы с данными абитуриента нажмите на кнопку "Данные абитуриентов" (откроется окно для работы с данными всех абитуриентов) или "Абитур. по данному направлению" (откроется окно для раоты с данными абитуриентов выбранного направления).
    Для быстрого выхода из системы: "Файл"-"Выход"
